PGMax Reference Documentation
==============================

PGMax is a library for working with Factor Graphs in
`JAX <https://jax.readthedocs.io/en/latest/>`_.
It currently provides an interface for specifying factor graphs of any type,
as well as an efficient implementation of max-product belief propagation
and inference on these graphs.
